# 1장 데이터베이스와 SQL

## 1강. 데이터베이스

데이터베이스 : 데이터의 집합

데이터 : 컴퓨터 안에 기록되어 있는 숫자 

데이터를 영구적으로 보관하기 위해, 하드디스크나 SSD등 비휘발성 저장장치에 저장 

### 1. 시스템 내의 데이터베이스

데이터베이스의 운용 

데이터 센터의 서버에서 운용 

온라인 쇼핑, POS 기기 

개인용 컴퓨터 or 휴대용기기 내장 

휴대전화의 전화번호부 

### 2. DB와 DBMS

DB : 데이터베이스 

DBMS : 데이터베이스 관리 시스템 

- 생산성 향상 : 데이터 검색, 추가, 삭제, 갱신 기능
- 기능성 : 복수 유저의 요청에 대응, 대용량의 데이터 저장, 고속 검색
- 신뢰성 : 여러대의 하드웨어를 통한 확장성과 부하 분산을 구현
    - 집어넣기 & 내보내기 기능 : 데이터베이스 백업

### 3. 데이터베이스를 조작하는 언어 SQL

DBMS와의 대화에 필요한 것 

관계형 데이터베이스 관리 시스템 조작시 사용 

SQL 명령 종류 

- DML : 데이터베이스 데이터 신규 추가, 삭제, 내용 갱신
- DDL :  데이터 베이스 객체 만들거나 삭제
- DCL :  트랜잭션을 제어 하는 명령과 데이터 접근권한 제어 명령

## 2강. 다양한 데이터베이스

관계형 데이터베이스  : SQL로 데이터를 다루는 데이터베이스 

### 1. 데이터베이스 종류

- 계층형 데이터베이스 : 폴더와 파일 등의 계층 구조로 데이터를 저장하는 방식의 데이터베이스
- 관계형 데이터베이스 :  행과 열을 가지는 표 형식 데이터를 저장하는 형태의 데이터베이스
    - 표 형식 데이터: 2차원 데이터 ( ex. 엑셀 시트)
- 객체지향 데이터베이스
    - 객체 그대로를 데이터베이스의 데이터로 저장하는 것
- XML 데이터베이스
    - 태그를 통해 마크업 문서를 작성할 수 있게 정의한 것
        - SQL 명령어 사용 불가
- 키 - 밸류 스토어
    - 키와 밸류의 조합,

### 2. RDBMS 사용 시스템

대부분의 메인시스템 - RDBMS 사용 

- 웹 시스템과의 연동
- 하드웨어에 내장

### 3. 데이터베이스 제품

RDBMS 소개 

- Oracle
- DB2
- SQL Server
- PostgreSQL
- MySQL
- SQLite

### 4. SQL의 방언과 표준화

고유 방언 : 특정 데이터베이스 제품에만 통용됨 

키워드 생략: DELETE (FROM 생략) 

외부결합 : 연산자 차이 

## 3강. 데이터베이스 서버

데이터베이스 = 네트워크로 연결된 서버 

일반적인 RDBMS  → 네트워크 상 하나의 서버를 두고 독점 사용 

자신의 PC에 클라이언트와 서버 모두 실행시켜 운용 

### 01. 클라이언트/서버 모델

클라이언트와 서버로 소프트웨어를 나누고, 복수의 컴퓨터 상에서 하나의 모델을 구현하는 시스템 

- 웹 시스템에서의 클라이언트/서버
    - 웹 시스템: 브라우저와 웹 서버로 구성되는 클라이언트/서버 모델 시스템
    - 사용자 → 페이지 열람 요청 ( 리퀘스트=요청 ) → 서버 html 데이터 클라이언트 반환 ( 리스폰스=응답)→ 화면 표시
- RDBMS의 클라이언트/서버
    - 사용자 인증 : 사용자별로 데이터베이스 접근 제한 (IDPW)
- SQL 명령 실행

### 02. 웹 애플리케이션의 구조

웹 애플리케이션 = 웹 서버 + 데이터베이스 서버 

클라이언트 = 브라우저, 서버 = 아파치, IIS 

웹 어플리케이션 = 데이터베이스 必 

- 제어용 프로그램
- CGL: 동적 컨텐츠를 위한 확장 방식

실제 데이터베이스 접속 : PHP, 루비 등의 프로그래밍 언어로 이루어진 CGI 프로그램 

데이터베이스 서버 사용

데이터베이스 서버 접속 → SQL 명령 전달 → 실행결과 클라이언트로 되돌아감 

이때, **CGL 프로그램이 데이터베이스의 클라이언트** 

⇒ 클라이언트/서버 모델은 유연한 하드웨어 구성을 실현한다 

### 03. MySQL 서버와 mysql 클라이언트

MySQL : 데이터베이스 서버 

mysql 커맨드 : 클라이언트 

루프 백 접속 : 클라이언트에서 서버에 접속할 때, 네트워크를 경유해서 PC의 서버로 되돌아오는 형태의 접속
